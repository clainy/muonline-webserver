# NyxWeb Server (TypeScript)

`NyxWeb muonline website - server`

## Create an .env file

```
PORT=server port
JWT_KEY=mySecretKey
DB_NAME=name
DB_USER=user
DB_PASS=password
DB_HOST=host
DB_PORT=port
```

## Additional columns

##### MEMB_INFO

```
jwt_token varchar(100)
admin_lvl int default 0
```
